## Struct_input_json_go

This is a simple Go program that allows you to enter note data and saves each new note in JSON format. The application prompts you for a note title and content, then stores the information for future use.

### Features
- Enter note title and content interactively
- Each note is saved as a JSON file
- Easy to use and extend

### Setup
To get started, simply clone the repository and run the program:

```
git clone https://github.com/bits-and-atoms/Struct_input_json_go.git
cd Struct_input_json_go
go run .
```

No additional setup is required.
